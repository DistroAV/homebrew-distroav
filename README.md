# homebrew-distroav

Install DistroAV easily from the command line on macOS.

## Installation

```sh
brew tap DistroAV/distroav
brew install --cask distroav
```

## Updating

```sh
brew upgrade --cask distroav
```

## Uninstalling

```sh
brew uninstall --cask distroav
```

## About

This Homebrew tap provides the [DistroAV](https://distroav.org) cask, an NDI® integration plugin for OBS Studio. The cask is automatically updated when a new release is published.
